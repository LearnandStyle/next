数据的删除
方法：
splice(index, num)
说明：
从索引index开始，删除num个元素，返回被删除元素组成的数组，原来的数据改变
数组的查询
方法：
slice(n, m)
说明：
从索引n开始找到索引为m处，不包含m，把找到的部分以一个新数组返回，原来的数组不变
其他：
slice(0)/slice()-->数组克隆，克隆一份和原来数组一模一样的数组、slice(n)-->从索引n开始找到末尾、slice支持负数索引-->浏览器解析的时候是按照总长度+负数索引 来处理的
扩展：
Array.prototype.mySlice = function mySlice(){ var n = arguments[0], m = arguments[1], newAry = []; n = n === undefined ? 0 : (isNaN(n) ? 0 : (n >= 0 ? (n < this.length ? Math.floor(Number(n)) : this.length) : (n < -this.length ? 0 : (n <= -1 ? this.length + Math.ceil(Number(n)) : 0)))); m = m === undefined ? this.length : (isNaN(m) ? 0 : (m >= 0 ? (m < this.length ? Math.floor(Number(m)) : this.length) : (m < -this.length ? 0 : this.length + Math.ceil(Number(m))))); while(n < m){ newAry[newAry.length] = this[n]; n++; } return newAry; };
数组的拼接
方法：
concat()
说明：
将多个数组进行拼接，把内容放在原来数组的后面（可以是数组，可以是一些数据值），返回拼接后的新数组，原来数组不变
其他：
concat()--> 什么都没拼接，相当于把原有数组克隆一份一模一样的新数组出来
数组转换成字符串
方法：
toString()
说明：
转换后的字符串，原有数组不变，把每一项以逗号分隔
数组转换成字符串
方法：
join()
说明：
把数组按照指定的分隔符转换为字符串，和字符串中的split()相对应，转换后的字符串，原有数组不变
其他：
join()-->参数不传，默认以逗号分隔
扩展：
已知数组中的每一项都是数字，实现数组求和！（eval(ary.join('+'))-->把字符串变为JS表达式执行）
数组中的排列
方法：
reverse()
说明：
把数组中的每一项倒过来排列，排列后的数组，原有数组改变
数组中的排序
方法：
sort()
说明：
参数无或者回调函数，排序后的数组，原有数组改变
其他：
sort()-->不传参数的情况下，可以给10以内的数字进行升序排列，多位数只识别第一位、sort(callback)-->（return a - b; --> 升序、return b - a;-->降序）
数组中是否包含某一项
方法：
indexOf()、lastIndexOf()
说明：
获取当前项在数组中第一次或者最后一次出现位置的索引，如果数组中没有这一项，返回索引-1
其他：
在IE6~8下不兼容、字符串中的这两个方法兼容所有的浏览器包括IE6~8
扩展：
Array.prototype.myIndexOf = function myIndexOf(value) { var result = -1; for(var i = 0; i < this.length; i++){ if(value === this[i]){ result = i; break; } } return result; }

数组去重
说明：
遍历数组中的每一项，把每一项和它后面的项依次比较，如果相同了，则把相同的这一项在原来的数组中删除即可
扩展：
for(var i = 0; i < ary.length - 1; i++){ var cur = ary[i]; for(var j = i + 1; j < ary.length;){ cur === ary[j] ? ary.splice(j, 1) : j++; } }
说明：
利用indexOf来验证当前数组中是否包含某一项，把包含的当前项删除
扩展：
for(var i = 0; i < ary.length; i++){var cur = ary[i];var curNextAry = ary.slice(i + 1);if(curNextAry.indexOf(cur) > -1){ary.splice(i, 1);i--;}}
说明：
对象键值对处理：遍历数组中的每一项，把每一项作为新对象的属性名和属性值存储起来；每一次存储之前验证一下当前对象中该属性是否存在，如果存在我们则不再存储并且把当前这个重复项删除掉即可；如果不存在我们就存储进来即可。
扩展：
Array.prototype.myUnique = function myUnique() {var obj = {};for(var i = 0; i < this.length; i++){var item = this[i];if(typeof obj[item] !== 'undefined'){this[i] = this[this.length - 1];this.length--;i--;continue;}obj[item] = item;}obj = null;return this;};
说明：
相邻比较法（首先给数组进行排序，然后相邻两项比较，相同的话，把后一项在数组中去掉）
冒泡排序
说明：
让数组中的当前项和后一项进行比较，如果当前项大于后一项，我们让两者交换位置（由小到大）
扩展：
function bubble(ary) {for(var i = 0; i < ary.length - 1; i++){for(var j = 0; j < ary.length - 1 - i; j++){if(ary[j] > ary[j + 1]){var temp = ary[j];ary[j] = ary[j + 1];ary[j + 1] = temp;}}}return ary;}

快速排序
说明：
找出中间项，把剩余项中的每一个值和中间项进行比较，比它小的放在左边（新数组），比它大的放在右边（新数组）
扩展：
function quick(ary) {if(ary.length <= 1){return ary;}var centerIndex = Math.floor(ary.length / 2);var centerValue = ary.splice(centerIndex, 1)[0];var aryLeft = [];var aryRight = [];for(var i = 0; i < ary.length; i++){var cur = ary[i];cur < centerValue ? aryLeft.push(cur) : aryRight.push(cur);}return quick(aryLeft).concat(centerValue, quick(aryRight));}
